  <name>Task 1: Alert dispatcher with signal change detection and message templates</name>
  <files>src/alerts/dispatcher.py, src/alerts/bot.py, tests/test_dispatcher.py</files>
  <behavior>
    - AlertDispatcher stores the last seen signal per mode (SAVER, TRADER)
    - check_and_alert(new_signal) compares against stored previous signal:
      - If recommendation changed → dispatch signal change alert
      - If confidence crossed ±20% threshold → dispatch confidence shift alert
      - If neither changed → no alert (debounce)
    - Signal change alert includes:
      - Old → New recommendation with emoji badges
      - New confidence level
      - Reasoning from new signal
      - Gap info (VND and %) if available
      - "⚠️ Đây là thông tin thị trường, không phải tư vấn đầu tư." disclaimer
    - Confidence shift alert includes:
      - Current recommendation (unchanged)
      - Old → New confidence with direction arrow (↑ or ↓)
      - Reasoning
      - Same disclaimer
    - Price movement detection:
      - Check if latest SJC bar price moved >2% from previous snapshot
      - Format price movement alert with old/new price and % change
      - Include disclaimer
    - All alerts sent to all chat_ids in SUBSCRIBED_CHATS via bot.send_message()
    - No subscribers → skip sending (log info)
    - Exceptions during send are caught and logged (don't crash pipeline)
  </behavior>
