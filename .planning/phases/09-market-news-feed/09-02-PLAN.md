# Plan 09-02: News Storage + API Endpoints + Admin Manual Entry

## Objective

Persist news articles in the database, provide API endpoints for querying news, and add admin endpoints for manually adding news items (e.g., State Bank announcements).

## Context

- Follow existing storage pattern: SQLAlchemy model + async repository functions
- Follow existing admin API pattern from `src/api/routes/admin.py` (policy events CRUD)
- News is display-only, not signal input
- Manual entries allow admins to add State Bank announcements

## Tasks

### Task 1: NewsItem database model + repository

**type:** auto
**tdd:** true

Create NewsItem SQLAlchemy model and repository functions.

**behavior:** A `news_items` table with columns: id, title, url (unique), source, published_at, excerpt, category, is_manual (bool, for admin entries), created_at. Repository functions: save_news_item(), get_recent_news(limit, category).

**implementation:** Add `NewsItem` model to `src/storage/models.py`. Add repository functions to `src/storage/repository.py`.

**done when:** Model creates table, save/get functions work via async session.

---

### Task 2: News API endpoints

**type:** auto
**tdd:** true

REST endpoints for querying news articles.

**behavior:** GET `/api/news` returns recent news items (default 20, configurable limit). Optional query params: `category` filter, `limit` (max 50). GET `/api/news/{id}` returns single article. Returns JSON with title, url, source, published_at, excerpt, category.

**implementation:** Create `src/api/routes/news.py` with FastAPI router. Wire into main.py.

**done when:** GET /api/news returns articles from DB, GET /api/news/{id} returns single item, 404 for missing.

---

### Task 3: Admin endpoint for manual news entry

**type:** auto
**tdd:** true

Admin POST endpoint for manually adding news (State Bank announcements, etc.).

**behavior:** POST `/api/admin/news` accepts: title, url (optional), source, published_at (optional, defaults to now), excerpt (optional), category (optional, defaults to "state_bank"). Sets is_manual=True. Returns created item.

**implementation:** Add to existing `src/api/routes/admin.py`.

**done when:** Admin can POST a news item, it appears in GET /api/news.
