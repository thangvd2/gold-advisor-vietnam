## Key Design Decisions

- **D-01**: Use yfinance with ticker "GC=F" (gold futures) for XAUUSD spot price. Fallback to "XAUUSD=X" if GC=F fails. Both are free.
- **D-02**: Use Vietcombank **selling rate** (market rate) for USD/VND conversion — NOT the official SBV rate. The market rate reflects actual import cost for gold shops. Vietcombank publishes rates at `https://www.vietcombank.com.vn/api/exrates/usd` (JSON endpoint — verify during implementation, fallback to scraping their rates page).
- **Conversion formula**: International gold is priced per troy ounce (oz). Vietnamese gold is priced per lượng (37.5 grams). 1 troy ounce = 31.1034768 grams, 1 lượng = 37.5 grams. So: 1 oz ≈ 1.09714286 lượng. Price in VND/lượng = (price_USD_per_oz × USD_VND_rate) / 1.09714286. Alternatively: price_VND_per_oz = price_USD_per_oz × USD_VND_rate, then price_VND_per_luong = price_VND_per_oz × (31.1034768 / 37.5).
- **Adapter pattern** (from ARCHITECTURE.md): Each data source extends `DataSource` ABC and returns `FetchedPrice`. The repository only sees `FetchedPrice` objects.
- **Pydantic models** for ingestion (FetchedPrice) separate from ORM models (PriceRecord). Repository converts between them.

## Vietcombank FX Rate

The Vietcombank API endpoint `https://www.vietcombank.com.vn/api/exrates/usd` returns JSON with buying/selling rates. Use the **selling rate** (giá bán) for conversion — this is the market rate at which gold shops buy USD to import gold. If this endpoint is blocked or returns unexpected data, fall back to:
1. `https://www.sbv.gov.vn` SBV rate (less accurate per D-02, but available)
2. A hardcoded last-known rate with a "stale" validation flag

## yfinance Details

- Ticker: "GC=F" (COMEX Gold Futures, most liquid gold proxy). Also try "XAUUSD=X" as fallback.
- Use `ticker.fast_info` for quick current price, or `ticker.history(period="1d")` for latest close.
- yfinance is synchronous — wrap in `asyncio.to_thread()` to avoid blocking the event loop.
- Handle `yfinance` exceptions (network errors, empty data, delisted ticker) gracefully — return None with error logging, don't crash.
