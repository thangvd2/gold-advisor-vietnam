**CRITICAL DATA SHAPE for gap calculation:**
- International gold: `source='yfinance'`, `product_type='xau_usd'`, `price_vnd` = VND per lượng (converted). Do NOT use `sell_price` (that's the previous close, not spot).
- Domestic SJC: `source IN ('sjc','pnj','doji','btmc','phuquy')`, `product_type='sjc_bar'`, `sell_price` = VND per lượng. `price_vnd` is NULL for domestic records.

       ```python
       import duckdb
       from pathlib import Path

       def get_duckdb_connection(db_path: str | Path) -> duckdb.DuckDBPyConnection:
           con = duckdb.connect()
           con.execute("INSTALL sqlite; LOAD sqlite")
           con.execute(f"ATTACH '{db_path}' AS db (TYPE sqlite, READ_ONLY)")
           return con
       ```
